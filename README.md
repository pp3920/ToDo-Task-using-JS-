This application is used to manage a to-do list, where users can add details such as Task Name, Category, Deadline, and Status. Once the information is provided, the task is added and displayed in the task list.

The application includes validation logic to ensure all required fields are filled. If any field is missing, the UI displays an error message prompting the user to complete the required inputs.

Additionally, the application contains logic to track task deadlines. If a task is not marked as completed and its deadline has passed based on the current business date, the system automatically updates its status to "Task is Overdue."
